Refresh all the STW <a href="https://shrinktheweb.com/">shrinktheweb.com</a> screenshots at once

== Description ==

This plugin works with the Directorypress theme, WP-Portfolio plugin, and/or with the <a href="http://wordpress.org/plugins/shrinktheweb-website-preview-plugin/">STW Wordpress plugin</a>. 
It refreshes all the thumbnails at once with the option to schedule the refresh action in selected intervals or immmediately.
It also allows the users to clear the cache immediately.

== Installation ==

1. Upload the content of wp-random-quote.zip to a dedicated folder in your `/wp-content/plugins/` directory.
2. Activate the plugin on the 'Plugins' page in WordPress.
